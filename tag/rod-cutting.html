动态规划笔记（1）——Rod cutting

Thu 08 March 2012
By Adoo
Tags: dynamic programming rod cutting 算法导论 笔记 In Introduction to Algorithm -third edition

动态规划（dynamic programming）,与“分治思想”有些相似，都是利用将问题分 为子问题，并通过合并子问题的解来获得整个问题的解。于“分治”的不同之处在 于，对于一个相同的子问题动态规划算法不会计算第二次，其实现原理是将每一 个计算过的子问题的值保存在一个表中。

动态规划一般被用于解决“最优解”问题。编写动态规划算法可以按照以下四个步 骤(引自原文)：

  2. Recursively define the value of an optimal solution.
