Windows 8 初体验

Fri 16 September 2011
By Adoo
Tags: c++ windows 8 xaml In Metro

几个月前，就已经对windows8很期待。主要是因为听说xaml将支持C++。14号刚放出下载链接就 赶去下下来，准备一探究竟。不过当天好哥们磊子过来了，于是和小庄同学，磊子我自己三个 人，一起到球场上肆意了一个下午，晚上喝酒聊天，已经忘掉这回事了。昨天又停电一天，到今 天才有机会体验下windows 8。 第一件事自然是打开Windows 8预览版自带的visual studio 2011,没有令我失望，可以使用C++来开发Metro程序。用xaml来为C++写界面，相信会是不一样的 感觉。之前一直用WPF来写界面，用C++/Cli来包装C++写好的代码，供C#调用。这实在是一种畸 形的工作方式，但是好的是，马上就要向其告别了。

但是实话说，频繁的在Metro与传统桌面像切换，确实另我有点不习惯，这倒是因为Metro程序太 少的原因。另外预览版中应用商店并没有开放。作为开发人员来讲，另外Metro程序无法关闭， 只能用Alt+F4。QQ2011的聊天窗口容易假死。性能方面倒是表现不错 ...
